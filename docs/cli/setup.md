# `aikaclaw setup`

Initialize `~/.aikaclaw/aikaclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
aikaclaw setup
aikaclaw setup --workspace ~/.aikaclaw/workspace
```

To run onboarding via setup:

```bash
aikaclaw setup --wizard
```
